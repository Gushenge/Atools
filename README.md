## Android实用工具合集-ATools

- ### 分为两个版本
     kotlin  
        `implementation 'com.gushenge.atools:kotlin:0.1.1' `  
     java  
        `implementation 'com.gushenge.atools:java:0.1.1'`
        
- ### 说明
    APP的targetSdkVersion应大于23  
    两个仓库的targetSdkVersion都为29
